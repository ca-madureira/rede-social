# Rede Social [em andamento]

## Sobre o Projeto
O projeto tem como objetivo permitir que o usuário possa se cadastrar, criar um perfil, fazer postagens, seguir, curtir e comentar postagens de outros usuários.  


### Frontend
- React Js
- Redux

### Backend
- Express
- Node Js
- JWT
- Mongoose
