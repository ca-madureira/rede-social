# Rede Social [em andamento]

## Sobre o Projeto
O projeto tem como objetivo:
-  Cadastrar usuário
-  Fazer login
-  Criar um perfil
-  Postar imagens ou textos
-  Seguir outros usuários
-  Curtir e comentar postagens de outros usuários



### Frontend
- React Js
- Redux

### Backend
- Express
- Node Js
- JWT
- Mongoose
